js 实现斐波那契数列

dyygtfx Updated at Apr 14, 2016

在数学上，费波那契数列是以递归的方法来定义：

F_0=0 F_1=1 Fn = F{n-1}+ F_{n-2}（n≧2）

用文字来说，就是费波那契数列由0和1开始，之后的费波那契系数就由之前的两数相加

用 js 通过递归来实现，代码如下：

function fib(num) {
  if (num === 1 || num === 2) return 1
  return fib(num-1)+fib(num-2)
}
